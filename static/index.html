📋 Quick Setup Guide

Step 1: Install Ollama

  • Go to https://ollama.ai/download
  • Download and install Ollama for Windows
  • Restart your computer after installation

Step 2: Start Ollama Service

  • Open Command Prompt or PowerShell
  • Run: ollama serve
  • Keep this window open while using SkDucky AI

Step 3: Install CodeLlama Model

  • Open another Command Prompt
  • Run: ollama pull codellama
  • Wait for download to complete (this may take a while)

Step 4: Test in SkDucky

  • Go back to the Chat page
  • Enable "Use Ollama AI" option
  • Ask SkDucky to generate some code!

🛠️ Automated Setup

You can also use our automated setup script:

setup_ollama.bat

This script will check if Ollama is installed, start the service, and install the CodeLlama model automatically.
